Introduction

Base URL: http://delivery.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Response Codes

Status code: 200(OK, request was successful) - Example Response
{
"success": true,
"data": [{}, {}],
"message": "Item(s) found"
}
Status code: 201(CREATED, data was saved successfully) - Example Response
{
"success": true,
"data": [{}, {}],
"message": "Item created successfully"
}
Status code: 202(ACCEPTED, update was successful) - Example Response
{
"success": true,
"data": [{}, {}],
"message": "Item updated successfully"
}
Status code: 202(NO_CONTENT, deletion was successful) - Example Response
Status code: 422(UNPROCESSABLE_ENTITY, validation error) - Example Response
{
"errors": {},
"message": "The name field is required. (and 4 more errors)"
}
Status code: 500(SERVER_ERROR, internal server error) - Example Response
{
"success": false,
"message": "An error occurred, please try again later"
}
Status code: 403(FORBIDDEN, the request is not authorized) - Example Response
{
"message": This action is unauthorized.,
}

Authenticating requests

This API is not authenticated.

Authentication

Register New User.

This endpoint allows a user to register

Example request:
curl --request POST \
    "http://delivery.test/api/v1/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"ivdesqwxazrqkpirpb\",
    \"email\": \"nthompson@example.net\",
    \"password\": \"odio\",
    \"password_confirmation\": \"cupiditate\"
}"

const url = new URL(
    "http://delivery.test/api/v1/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "ivdesqwxazrqkpirpb",
    "email": "nthompson@example.net",
    "password": "odio",
    "password_confirmation": "cupiditate"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

import json

Filter: /restaurants?filter[name]=kofibusy lounge [can be filtered by name and or location]. Sort: /restaurants?sort=name(Ascending) or restaurants?sort=-name (Descending) [can be ordered by name and location].

Example request:
curl --request GET \
    --get "http://delivery.test/api/v1/restaurants" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"

    "http://delivery.test/api/v1/restaurants"
